# <a name="default-subkey"></a>Subchave padrão
O registro pode especificar uma fonte de dados padrão com a subchave padrão. Essa subchave é um caso especial de uma subchave de especificação de fonte de dados e tem os mesmos valores como qualquer outro subchave de especificação do código-fonte dados. A única diferença é que ele não está listado como um valor sob a subchave de fontes de dados ODBC.

# <a name="date-arithmetic"></a>Aritmética de data
O driver dá suporte à adição e subtração de um número inteiro de uma coluna de data, hora ou carimbo de hora. Para uma coluna de data, o inteiro que especifica o número de dias para adicionar ou subtrair. Para uma coluna de hora ou carimbo de hora, o inteiro que especifica o número de segundos para adicionar ou subtrair.

# <a name="odbc-component-files"></a>Arquivos de componentes ODBC
Os arquivos de Driver de ODBC são necessários para cada componente do banco de dados de área de trabalho de ODBC Drivers 4.0.  
  
 Alguns arquivos são necessários por mais de um componente. Esses arquivos devem ser redistribuídos, se você pretende redistribuir quaisquer dos Drivers ODBC de banco de dados de área de trabalho com seus aplicativos comerciais.  
  
 Esses arquivos são instalados no diretório \Windows\System para o Windows 95 ou posterior, ou no diretório \Windows\System32 para Windows NT 3.51 ou posterior e Windows 2000. Se os arquivos ODBC foram instalados anteriormente em um diretório diferente, certifique-se de que você use o diretório de arquivos \Windows\System (ou System32) mais recente.

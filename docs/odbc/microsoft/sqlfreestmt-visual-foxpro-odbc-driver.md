# <a name="sqlfreestmt-visual-foxpro-odbc-driver"></a>SQLFreeStmt (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas de Driver ODBC do Visual FoxPro. Para obter informações gerais sobre essa função, consulte o tópico apropriado sob [referência da API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte a: Completo  
  
 Conformidade com a API ODBC: Nível de núcleo  
  
 Interrompe o processamento associado a um determinado *hstmt*, fechará quaisquer cursores abertos associados a *hstmt*, descartará resultados pendentes e, opcionalmente, libera todos os recursos associados à instrução identificador.  
  
 Para obter mais informações, consulte [SQLFreeStmt](../../odbc/reference/syntax/sqlfreestmt-function.md) na *referência do programador de ODBC*.

# <a name="position-method-sqlservernclob"></a>Método position (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera a posição do caractere do objeto **NClob** ou da subcadeia de caracteres especificada no **NClob** com base na posição inicial.

|Nome|Descrição|  
|----------|-----------------|  
|[Método position &#40;java.sql.NClob, long&#41;](../../../connect/jdbc/reference/position-method-java-sql-nclob-long.md)|Recupera a posição do caractere na qual o objeto **NClob** especificado *searchstr* aparece nesse objeto **NClob** .|  
|[método &#40;de posição Java. lang. String,&#41; &#40;Long SQLServerNClob&#41;](../../../connect/jdbc/reference/position-method-java-lang-string-long-sqlservernclob.md)|Recupera a posição do caractere na qual a subcadeia de caracteres especificada *searchstr* aparece no valor **NClob** representado por esse objeto **NClob** .|

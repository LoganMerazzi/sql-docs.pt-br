# <a name="getshort-method-sqlservercallablestatement"></a>Método getShort (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o valor do parâmetro designado como um **short** na linguagem de programação Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Descrição|  
|----------|-----------------|  
|[getShort (int)](../../../connect/jdbc/reference/getshort-method-int.md)|Recupera o valor do parâmetro designado como um **short** na linguagem de programação Java, considerando o índice do parâmetro.|  
|[getShort (java.lang.String)](../../../connect/jdbc/reference/getshort-method-java-lang-string.md)|Recupera o valor do parâmetro designado como um **short** na linguagem de programação Java, considerando o nome do parâmetro.|

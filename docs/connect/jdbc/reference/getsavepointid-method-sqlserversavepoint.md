# <a name="getsavepointid-method-sqlserversavepoint"></a>Método getSavepointId (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Obtém a ID do ponto de salvamento.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public int getSavepointId()  
```  
  
## <a name="return-value"></a>Valor retornado  
 Uma **int** valor.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Esse método getSavepointId é especificado pelo método getSavepointId na interface do savepoint.

# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  A interface IBCPSession2 é uma extensão para IBCPSession que fornece uma função de membro que é uma alternativa para chamar IBCPSession:: BCPColFmt para cada coluna.  IBCPSession2 herda de IBCPSession e adiciona um novo método: [IBCPSession2:: BCPSetBulkMode](../../oledb/ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).

# <a name="dbosystargetservergroups-transact-sql"></a>dbo.systargetservergroups (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  Registra quais grupos de servidores de destino estão atualmente inscritos neste ambiente multisservidor.  
  
  
|Nome da coluna|Tipo de dados|Descrição|  
|-----------------|---------------|-----------------|  
|**servergroup_id**|**int**|ID de grupo de servidores|  
|**name**|**sysname**|Nome do grupo de servidores|

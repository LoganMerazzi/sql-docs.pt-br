# <a name="mdx-data-definition---drop-kpi"></a>Definição de dados MDX – DROP KPI


  Elimina o KPI (indicador chave de desempenho) especificado do cubo mencionado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
DROP KPI CURRENTCUBE | Cube_Name.KPI_Name   
```  
  
## <a name="arguments"></a>Argumentos  
 *Cube_Name*  
 Uma cadeia de caracteres válida que especifica o nome do cubo.  
  
 *KPI_Name*  
 Uma cadeia de caracteres válida que especifica o nome do KPI a ser eliminado.

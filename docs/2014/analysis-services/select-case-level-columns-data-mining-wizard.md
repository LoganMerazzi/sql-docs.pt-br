# <a name="select-case-level-columns-data-mining-wizard"></a>Selecionar colunas de nível de caso (Assistente de Mineração de Dados)
  Use a página **Selecionar Colunas de Nível de Caso** para selecionar os atributos e medidas da dimensão do cubo de origem adjacente que serão as colunas de nível de caso da estrutura de mineração.  
  
> [!NOTE]  
>  Essa página será exibida apenas se você tiver selecionado **De cubo existente** na página **Selecionar o Método de Definição** do assistente.  
  
 **Para obter mais informações:** [Assistente de mineração de dados &#40;Analysis Services - mineração de dados&#41;](data-mining/data-mining-wizard-analysis-services-data-mining.md), [criar uma estrutura de mineração relacional](data-mining/create-a-relational-mining-structure.md)  
  
## <a name="options"></a>Opções  
 **Atributos e medidas relacionados**  
 Selecione os atributos e medidas que serão as colunas de nível de caso da estrutura de mineração. Todos os atributos selecionados devem estar em hierarquias ou ter suas hierarquias de atributo habilitadas.

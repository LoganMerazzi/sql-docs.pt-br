# <a name="name-property-securitycertificate-class"></a>Propriedade Name (classe SecurityCertificate)
  Obtém o nome do certificado de segurança.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
object  
.Name [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Um objeto da [classe SecurityCertificate](securitycertificate-class.md) que representa um certificado de segurança.  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/Valor do retorno  
 Um valor da cadeia de caracteres que especifica o nome que emitiu o certificado de segurança.

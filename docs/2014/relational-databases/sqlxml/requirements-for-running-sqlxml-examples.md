# <a name="requirements-for-running-sqlxml-examples"></a>Requisitos para executar exemplos do SQLXML
  O [programação do SQLXML 4.0](sqlxml-4-0-programming-concepts.md) seção fornece vários exemplos. Este tópico lista os requisitos para criar exemplos funcionais.  
  
 Para criar os exemplos funcionais a partir de exemplos do SQLXML 4.0, você precisa ter estes programas instalados:  
  
-   O banco de dados de exemplo AdventureWorks. Para obter mais informações, consulte [bancos de dados de exemplo AdventureWorks](http://msftdbprodsamples.codeplex.com/).  
  
-   Microsoft [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client. Para obter mais informações, consulte [Instalando o SQL Server Native Client](../native-client/applications/installing-sql-server-native-client.md).  
  
-   MDAC 2.6 ou posterior  
  
 Em muitos exemplos, são usados modelos para especificar consultas XPath com relação ao esquema XSD de mapeamento. Para obter mais informações, consulte [usando esquemas de XSD anotados em consultas &#40;SQLXML 4.0&#41;](annotated-xsd-schemas/using-annotated-xsd-schemas-in-queries-sqlxml-4-0.md).

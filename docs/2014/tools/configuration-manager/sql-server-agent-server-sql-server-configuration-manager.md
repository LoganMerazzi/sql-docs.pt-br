# <a name="sql-server-agent-ltservergt-sql-server-configuration-manager"></a>SQL Server Agent &lt;servidor&gt; (SQL Server Configuration Manager)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-winonly](../../includes/appliesto-ss-xxxx-xxxx-xxx-md-winonly.md)]
  Este é o serviço que dá suporte ao [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Agent. Para exibir as propriedades do serviço, clique nele com o botão direito do mouse no painel de detalhes e clique em **Propriedades**.  
  
 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Agent é um serviço do [!INCLUDE[msCoName](../../includes/msconame-md.md)] Windows que executa tarefas administrativas agendadas, chamadas *trabalhos*. Para obter mais informações sobre o [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Agent, consulte "Sobre o SQL Server Agent" nos Manuais Online do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .

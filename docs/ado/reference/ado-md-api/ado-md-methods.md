# <a name="ado-md-methods"></a>Métodos do ADO MD

|||  
|-|-|  
|[Fechar](../../../ado/reference/ado-md-api/close-method-ado-md.md)|Fecha um conjunto de células aberto.|  
|[GetSchemaObject](../../../ado/reference/ado-md-api/getschemaobject-method-ado-md.md)|Recupera um objeto de esquema do ADO MD (dimensão, hierarquia, nível ou membro) por seu nome exclusivo.|

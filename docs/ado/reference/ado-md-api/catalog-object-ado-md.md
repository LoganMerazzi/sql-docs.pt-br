# <a name="catalog-object-ado-md"></a>Objeto Catalog (ADO MD)
Contém multidimensional informações de esquema (ou seja, cubos e subjacente dimensões, hierarquias, níveis e membros) específicas para um provedor de dados multidimensional (MDP).  
  
## <a name="remarks"></a>Comentários  
 Com as coleções e propriedades de um **catálogo** do objeto, você pode fazer o seguinte:  
  
-   Abra o catálogo, definindo o [ActiveConnection](../../../ado/reference/ado-md-api/activeconnection-property-ado-md.md) propriedade a um padrão ADO [Conexão](../../../ado/reference/ado-api/connection-object-ado.md) objeto ou uma cadeia de caracteres de conexão válida.  
  
-   Identificar o **catálogo** com o [nome](../../../ado/reference/ado-md-api/name-property-ado-md.md) propriedade.  
  
-   Iterar os cubos em um catálogo usando o [CubeDefs](../../../ado/reference/ado-md-api/cubedefs-collection-ado-md.md) coleção.

# <a name="putoledbcommand-method"></a>Método put_OLEDBCommand
Esse método não realiza nenhuma operação e sempre retorna S_OK.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
HRESULT put_OLEDBCommand(  
      IUnknown *pOLEDBCommand  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 *pOLEDBCommand*  
 [in] Ponteiro para um objeto de comando OLE DB.  
  
## <a name="applies-to"></a>Aplica-se a  
 [IADOCommandConstruction](https://msdn.microsoft.com/library/windows/desktop/aa965677.aspx)

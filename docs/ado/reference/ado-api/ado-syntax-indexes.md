# <a name="ado-syntax-indexes"></a>Índices de sintaxe ADO
A sintaxe para chamar as propriedades e métodos do ADO difere dependendo do ambiente de desenvolvimento. O restante da referência da linguagem ADO usa o Microsoft Visual Basic linguagem de programação para ilustrar a sintaxe de método e propriedade ADO. No entanto, consulte as seções a seguir para exemplos de sintaxe mais específicos com base em sua linguagem de programação e a metodologia de:  
  
-   O [ADO para índice de sintaxe do Visual C++ para COM](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-for-com.md) aborda os métodos e propriedades do ADO sem usar o **#import** diretiva do compilador com o Microsoft Visual C++.  
  
-   O [ADO para índice de sintaxe do Visual C++ com #import](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-with-sharpimport.md) aborda os métodos e propriedades do ADO ao usar o **#import** diretiva do compilador com o Microsoft Visual C++.

# <a name="type-property-column-adox"></a>Propriedade Type (Column) (ADOX)
Indica o tipo de dados de uma coluna.  
  
## <a name="settings-and-return-values"></a>As configurações e valores de retorno  
 Define ou retorna um **longo** valor que pode ser um dos [DataTypeEnum](../../../ado/reference/ado-api/datatypeenum.md) constantes. O valor padrão é **adVarWChar**.  
  
## <a name="remarks"></a>Comentários  
 Essa propriedade é leitura/gravação até a [coluna](../../../ado/reference/adox-api/column-object-adox.md) objeto é anexado a uma coleção ou a outro objeto, após o qual ele é somente leitura.

# <a name="sortorderenum"></a>SortOrderEnum
Especifica a sequência de classificação para uma coluna indexada.  
  
|Constante|Valor|Descrição|  
|--------------|-----------|-----------------|  
|**adSortAscending**|1|Padrão. A sequência para a coluna de classificação é crescente.|  
|**adSortDescending**|2|Sequência de classificação para a coluna em ordem decrescente.|
